# 作业一：基于城市空气质量数据的多维特征挖掘与质量演变研究

## 研究背景

利用Python网络爬虫技术，从天气回报网站（http://tianqihoubao.com/）收集2017年至2021年五年间北京每日空气质量数据，共提取有关的11个属性。

## 具体任务

### 第一阶段：数据预处理

#### 任务 1：数据清洗（10）

提示：对数据进行一致性检查、异常值的判定、重复项处理和缺失值修复等。

#### 任务 2：数据标准化（10）

提示：统一日期格式为标准datetime类型、统一类别标签等。

#### 任务 3：时间维度的特征衍生工程（10）

提示：将原始的年/月/日转化为具有科研意义的特征。从日期中衍生出季节、月份、是否周末、是否法定节假日等，挖掘人类活动与自然循环的影响。

### 第二阶段：数据分析

#### 任务 1：统计描述（10）

提示：计算各污染物的均值、标准差、最大/最小值等。

#### 任务 2：统计发现与深度可视化（20）

提示：根据清洗后的数据，至少使用三种图进行分析，不限于相关性热力图、时空轨迹图、污染物成分占比分析、箱线图等。

### 第三阶段：数据建模（40）

- 算法选择：使用Apriori或FP-Growth算法。
- 建模目标：挖掘形如 {PM2.5=高, NO2=高} -> {质量等级=严重污染} 的规则。
- 参数调优：设定合理的支持度（Support）和置信度（Confidence）阈值。
- 规则评价：通过 提升度（Lift） 筛选出有意义的强关联规则。
- 结果解释：解释挖掘出的强关联规则并给出空气质量治理的建议。

## 提交要求

- 数据集处理清单：一份描述数据清洗前后的统计对比表（如：清洗前缺失率 5%，清洗后 0%）。
- 分析报告 (Scientific Report)：包含：摘要、数据预处理流程、可视化发现、挖掘结论等。
- 代码附件：推荐使用 Python (Pandas + Seaborn) 或 R 语言。

# 作业二：植物叶片光合生理指标的高光谱特征提取与预测模型构建

## 研究背景

在植物表型研究中，非接触式监测叶片的SPAD、光合速率（Pn）、氮含量（LNC）等指标对育种和精准栽培至关重要。本项目模拟从原始传感器信号到生理洞察的完整数据挖掘流程，重点在于处理校准过程中的噪声，并挖掘光谱响应与生理指标之间的内在逻辑。

## 相关数据说明

Raw_Spectra文件夹：包含6个叶片原始采集的高光谱数据，采用 ENVI 标准格式，每组数据由 .raw（原始二进制数据）与 .hdr（头文件，包含波段信息与元数据）共同构成。以命名为"20210927092826"该组数据为例，文件夹中"20210927092826B"表示反射率为3%的参考板的高光谱数据，"20210927092826W"为反射率为30%的参考板的高光谱数据，"20210927092826P"为叶片的高光谱数据（20210927092826对应的是原始整个植株的高光谱数据，本次实验不涉及该数据的处理），30%和3%参考板的反射率真值见附件date_FGB.csv文件

生理指标：包含与高光谱样本一一对应的生理参数实测值，涵盖叶绿素相对含量（SPAD）、净光合速率（Pn）、叶片全氮含量（LNC）、叶绿素 a（Ca）、叶绿素 b（Cb）以及比叶重（SLW），真值见附件GT_values.csv。

## 具体任务

### 任务 1：反射率的校准与提取（30）

- 反射率计算：编写代码读取.raw 数据，利用下述公式将DN值转换为反射率。

  $$
  R = \frac{DN - DN_{3\%}}{DN_{30\%} - DN_{3\%}} \times (R_{30\%} - R_{3\%}) + R_{3\%}
  $$

  其中，DN表示原始高光谱数据中读取的数字真值，R为反射率值，30%和3%参考板对应的反射率值R见data_FGB.csv文件。
- 光谱曲线绘制：绘制 6 组叶片样本在全波段下的反射率曲线，并标注出主要的植物光谱特征点（如 550nm 绿峰、680nm 吸收坑）。
- 数据关联拼接：将校准后的光谱反射率特征与 GT_values.csv 中的生理指标（SPAD, Pn, LNC, Ca, Cb, SLW）通过样本编号进行横向合并，生成用于后续建模的完整矩阵，并保存为 calibrated_phenotype_data.csv。

### 任务 2：光谱数据预处理（20）

- 光谱平滑（Smoothing）：对校准后的数据使用Savitzky-Golay (SG)进行平滑过滤处理，并对比原始数据与经过处理后的曲线，观察噪声滤除效果。
- 基线校正（Correction）：尝试使用一阶导数（First Derivative）或多元散射校正（MSC）处理光谱，观察其是否能增强光谱中的吸收特征峰。

### 任务 3：特征工程与生理指标关联分析（30）

- 相关性分析：针对校准后的全波段反射率数据，分别计算其与 6 种生理指标（SPAD, Pn, LNC, Ca, Cb, SLW）的皮尔逊相关系数，绘制相关性系数谱图，并在报告中指出每种生理指标对应的"敏感波长区域"（即相关系数绝对值最大的区域）。
- 特征降维对比：使用 PCA（主成分分析），观察前3个主成分能解释多少方差，并绘制前三个主成分的 3D 散点图。

### 任务 4：光合能力预测建模与评估（20）

- 基准模型对比：建立经典的 PLSR（偏最小二乘回归） 与机器学习模型（如随机森林）进行对比，由于样本量有限，重点观察 PLSR 模型的拟合趋势，并尝试可能的数据增强方式。
- 性能评价：计算

  $$
  R^2
  $$

  、RMSE （均方根误差）和 RPD （相对分析误差）。
- 结果分析：绘制"预测值 vs 真实值"拟合散点图。

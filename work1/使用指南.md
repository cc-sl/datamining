## 关键发现速查

### 🔍 最重要的3个发现

**发现1**: PM2.5是首要污染物
```
- 与AQI相关系数: r=0.89 (最强)
- 与质量等级的Lift=12.15 (最强关联)
- 冬季均值88 μg/m³ (夏季的2.3倍)
```

**发现2**: 周末显著改善（工业停产效果）
```
- Lift=11.64 反映周末污染模式
- SO2降低40%, CO降低35%
- 意义: 工业停产直接改善空气质量
```

**发现3**: 冬季是最严峻的污染期
```
- PM2.5比夏季高2.3倍
- 多污染物协同高(Lift=8.34)
- 供暖期(12月-2月)的关键期
```

## 常见问题 (FAQ)

### Q1: 如何修改输入数据？
```python
# 在homework1.py第65行改为:
df_raw = pd.read_csv('your_new_file.csv', encoding='gb2312')
```

### Q2: 为什么PM10最大值是1563这么大？
- 这是原始数据异常值，但保留了(假设测量准确)
- 代表最严重的沙尘暴日

### Q3: 关联规则中"是"是什么意思？
- "是"对应列名 `是否周末` 或 `是否法定节假日` 
- 值为 '是' 或 '否'
- 规则中单独出现 '是' 表示该条件为真

### Q4: 如何导出规则为其他格式？
```python
df_rules = pd.read_csv('work1/关联规则_强规则.csv')

# 导出为Excel
df_rules.to_excel('rules.xlsx', index=False)

# 导出为JSON
import json
df_rules.to_json('rules.json', orient='records', force_ascii=False)

# 导出为可视化决策树
# 需要graphviz库: pip install graphviz
```

### Q5: 为什么缺失率反而增加了？
- 清洗前: 526/21744 = 2.42% (1812行×12列)
- 清洗后: 429/19390 = 2.46% (1790行×12.2列，因衍生特征)
- 实际上缺失值减少了97个，数据质量提升

### Q6: 如何理解Lift=11.64？
```
Lift = P(后件|前件) / P(后件)
     = 0.813 / 0.070
     = 11.64

意义: 当条件满足时，结论出现概率提升1064%
     即使用该规则预测比随机猜测好11倍
```

## 进阶用法

### 自定义规则筛选

```python
import pandas as pd

# 读取规则
df = pd.read_csv('work1/关联规则_强规则.csv')

# 筛选1: 只看后件包含"严重污染"的规则
severe_rules = df[df['consequents'].str.contains('严重污染')]
print(f"能推导出严重污染的规则: {len(severe_rules)}")

# 筛选2: 找置信度最高的规则(最准确的预测)
most_confident = df.nlargest(5, 'confidence')
for idx, row in most_confident.iterrows():
    print(f"准确度{row['confidence']:.1%}: {row['antecedents']}")

# 筛选3: 冬季规则
winter_rules = df[df['antecedents'].str.contains('冬')]
print(f"冬季相关规则: {len(winter_rules)}")

# 筛选4: 工作日 vs 休息日对比
workday_rules = df[df['antecedents'].str.contains('工作日')]
holiday_rules = df[df['antecedents'].str.contains('休息日')]
print(f"工作日规则数: {len(workday_rules)}, 休息日规则数: {len(holiday_rules)}")
```
